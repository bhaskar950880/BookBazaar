# 📚 BookBazaar

A Flipkart-style Full Stack Online Book Store built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## 🚀 Features

### 👤 Authentication
- Buyer Login/Register
- Seller Login/Register
- JWT Authentication

### 📚 Books
- Dynamic Book Listing
- Product Details Page
- Search & Filters
- Category Filtering

### 🛒 Cart System
- Add To Cart
- Remove From Cart
- Checkout Flow
- Real-time Total

### 📦 Orders
- Place Orders
- My Orders Page
- Order Tracking

### 🛍 Seller Dashboard
- Add Books
- Delete Books
- Manage Inventory
- Revenue Analytics

## 💻 Tech Stack

Frontend:
- HTML
- CSS
- Bootstrap
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Authentication:
- JWT
- bcryptjs
